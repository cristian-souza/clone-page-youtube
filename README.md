# Clone da Página de Vídeo do YouTube

Este projeto é um clone da página de vídeo do YouTube criado com HTML e CSS.

## Visão Geral

O projeto recria a aparência de uma página de vídeo do YouTube, incluindo:

-   Um player de vídeo embutido
-   O título do vídeo, informações do canal e botões de ação (curtir, não curtir, compartilhar, baixar)
-   A descrição do vídeo
-   Uma lista de vídeos sugeridos

## Estrutura de Arquivos

-   `index.html`: A estrutura principal da página web.
-   `assets/css/global.css`: Estilos globais e resets.
-   `assets/css/header.css`: Estilos para o cabeçalho.
-   `assets/css/main.css`: Estilos para o conteúdo principal.
-   `assets/images/`: Contém todas as imagens usadas no projeto.

## Como Executar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` em seu navegador.
